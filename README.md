# TeamTrack

A task management app built for college project teams — helps teams assign tasks, 
track status, and stay accountable to deadlines.

## Status
🚧 In development — backend setup in progress.

## Tech Stack
- Backend: Node.js, Express, MongoDB (Mongoose)
- Auth: JWT
- Frontend: React (coming soon)

## Getting Started (Backend)

1. Clone the repo
2. `cd backend`
3. `npm install`
4. Copy `.env.example` to `.env` and fill in your own values
5. `npm start` (or `node server.js`)

## Why This Project
Built to solve a real gap in how college project teams track task ownership 
and deadlines — most groups end up losing track of who owes what before a 
submission.
